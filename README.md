# Finance-Class
Laporan keuangan kelas dibuat untuk memastikan setiap pemasukan dan pengeluaran dana kelas tercatat dengan jelas, rapi &amp; transparan. Login ke akun-Masuk ke menu Dashboard-Tambah data pemasukan-Tambah data pengeluaran-Cek riwayat transaksi-Lihat laporan keuangan-(rekap saldo, pemasukan, pengeluaran)-Unduh atau cetak laporan-Logout setelah selesai.
